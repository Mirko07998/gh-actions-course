# Simple Calculator

## Overview
This is a simple command-line based calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Getting Started
You can run the application by executing the following command in your terminal:

```bash
python calculator.py
```

## Features
- Addition, subtraction, multiplication, and division operations.
- Error handling for non-numeric inputs and unsupported operators.
- Graceful exit and repeat functionality.

## Usage
1. Run the application and enter two numbers along with an operator when prompted.
2. The result will be displayed.
3. You can choose to perform another calculation or exit the application.

## Exit
The application terminates with a "Goodbye!" message.
