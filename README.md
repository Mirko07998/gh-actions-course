# Simple Calculator

This is a simple calculator application that allows users to perform basic arithmetic operations from the command line. It supports addition, subtraction, multiplication, and division, with proper error handling to ensure a smooth user experience.

## Features

1. Performs addition, subtraction, multiplication, and division.
2. Handles non-numeric input and unsupported operations gracefully.
3. Prevents division by zero.
4. Prompts user for repeated calculations.

## How to Run

To run the application, ensure you have Python installed and execute the following command in your terminal:

```
python calculator.py
```

## Error Handling

The application will display messages for:
- Non-numeric inputs
- Unsupported operators
- Division by zero

## Exit Message

After completing calculations, the user can choose to exit the program, at which point a goodbye message will be displayed.
