# Simple Calculator

## Overview
This is a simple calculator application that can be run from the terminal using Python. It supports basic arithmetic operations: addition, subtraction, multiplication, and division.

## How to Run
Run the application using the following command:
```
python calculator.py
```

## Features
- User-friendly interface
- Supports basic operations: +, -, *, /
- Error handling for invalid inputs and division by zero
- Ability to repeat calculations

## Testing
To run tests, use:
```
python -m unittest calculator/tests.py
```
