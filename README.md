# Simple Calculator

This is a simple command-line calculator application.

## Requirements

- Python 3.x

## How to Run

This application can be run from the terminal using the following command:

```
python calculator.py
```

## Features

- Addition, Subtraction, Multiplication, and Division operations.
- Error handling for non-numeric inputs, unsupported operators, and division by zero.
- Option to repeat calculations or exit the application.
