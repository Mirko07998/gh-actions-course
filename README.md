# Simple Calculator

This is a simple command-line calculator application built in Python. 

## How to Run

You can run the application from the terminal using the following command:

```
python calculator.py
```

## Features
- Supports basic arithmetic operations: addition, subtraction, multiplication, and division.
- Handles user input with proper validation and error messages.
- Allows the user to perform multiple calculations in a single session.
- Displays clear results and error messages to the user.

Enjoy using the Simple Calculator!
