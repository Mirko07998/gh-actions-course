# Simple Calculator Web App

## Description
This is a simple web application built with FastAPI that allows users to perform basic arithmetic calculations.

## Features
- Perform addition, subtraction, multiplication, and division.
- User-friendly web interface.
- Error handling for invalid inputs.

## How to Install Dependencies
Run the following command to install the required packages:
```
pip install -r requirements.txt
```

## How to Run the Server
To start the FastAPI server, run:
```
uvicorn main:app --reload --port 8001
```
Access the application in your web browser at `http://127.0.0.1:8001/`.

## Example Usage
1. Enter the first number.
2. Choose an operator.
3. Enter the second number.
4. Press the "Calculate" button to see the result.
