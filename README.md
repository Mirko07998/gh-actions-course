# Simple Calculator

## Overview
This is a simple command-line calculator that performs basic arithmetic operations including addition, subtraction, multiplication, and division.

## Getting Started
To run the application, open your terminal and navigate to the directory where `calculator.py` is located. Use the following command:

```
python calculator.py
```

## Features
- Supports addition, subtraction, multiplication, and division of two numbers.
- Handles user input and converts them from strings to numbers.
- Provides error messages for invalid inputs and operations.
- Asks the user if they want to perform another calculation.
