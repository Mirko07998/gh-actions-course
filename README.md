# Simple Calculator Web App

## Description
This is a full-stack Django web application that serves as a simple calculator, allowing users to perform basic arithmetic operations.

## Setup

1. Clone the repository:
   ```
   git clone <repository-url>
   cd calculator_project
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the development server:
   ```
   python manage.py runserver
   ```
   Access the application at `http://127.0.0.1:8000/`.

## Features

- Perform addition, subtraction, multiplication, and division.
- User-friendly error messages.
- Handles invalid inputs and division by zero.
- Continuous usage without page reload for repeated calculations.

## Example Usage

- Enter the first number, select an operator, enter the second number, and click "Calculate".
- Receive the result or an appropriate error message.
