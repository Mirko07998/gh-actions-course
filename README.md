# Simple Calculator

This project is a simple calculator that can perform basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features
- Add, subtract, multiply, and divide two numbers.
- Error handling for invalid input and operations.
- Ability to perform calculations repeatedly until the user decides to exit.

## Usage
To run the application, use the following command in your terminal:

```bash
python calculator.py
```

Follow the prompts to enter two numbers and an operator to perform a calculation.

## Running Tests
To run the test suite, use:
```bash
python -m unittest test_calculator.py
```
